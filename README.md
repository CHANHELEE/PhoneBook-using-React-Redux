# PhoneBook-using-React-Redux  
  
  1. 왼쪽에는 연락처 등록폼, 오른쪽에는 연락처 리스트와 search창이 있다.  
  
  2. 폼 입력후 , 'create'버튼을 누르면 오른쪽 연락처에 추가가 된다.(몇개의 연락처가 있는지 확인할 수 있다.)  

  3. 이름 입력후 'search'버튼을 누르면 해당이름을 검색한다.  

  4. 삭제 버튼을 누르면 삭제 된다.
